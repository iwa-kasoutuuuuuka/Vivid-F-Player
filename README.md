# Vivid F Player

モダンなデザインと使いやすさを追求した、Android用ビデオプレイヤーアプリです。
複数のフォルダを管理し、ファイル名順での連続再生やレジューム再生に対応しています。

## 🚀 主な機能

- 📁 **マルチフォルダ管理**: 複数の動画保存場所を登録・管理。
- 🔄 **自然順連続再生**: ファイル名の数字を考慮したインテリジェントな連続再生。
- 📺 **ピクチャー・イン・ピクチャー (PiP)**: 他のアプリを使いながら動画を視聴。
- 🎧 **バックグラウンド音声再生**: 画面を閉じても音声のみ継続可能（設定で切替）。
- 🎨 **Vivid & Premium UI**: グラスモーフィズムと鮮やかなグラデーションを採用したモダンなデザイン。
- 🖐️ **ジェスチャーコントロール**: 明るさ、音量、シークを直感的に操作。
- ⚡ **再生速度変更**: 0.5xから2.0xまで調整可能。
- 🔖 **レジューム再生**: 続きから再生。

## 📁 フォルダ構成

```text
f:/app/Android/
├── app/
│   ├── src/
│   │   ├── main/
│   │   │   ├── java/com/example/videoplayer/
│   │   │   │   ├── data/          # データモデル、リポジトリ、履歴管理
│   │   │   │   ├── player/        # ExoPlayerの管理、バックグラウンド再生サービス
│   │   │   │   ├── ui/            # アクティビティ、フラグメント、UIコンポーネント
│   │   │   │   └── util/          # 自然順ソートなどのユーティリティ
│   │   │   └── res/               # レイアウト、アイコン、テーマ設定
│   └── build.gradle.kts           # モジュールレベルのビルド設定
├── build.gradle.kts               # プロジェクトレベルのビルド設定
└── settings.gradle.kts            # プロジェクト設定
```

## 🛠 セットアップとビルド

1.  Android Studio を開き、本プロジェクトをインポートします。
2.  プロジェクト同期（Gradle Sync）を完了させます。
3.  `Run` ボタンを押して実機またはエミュレータで実行します。
